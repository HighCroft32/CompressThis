Project Title
Using the System.IO.Compression features introduced with dotNet 4.5, this application provides an executable for use in any later windows environment allowing compression of any number of files and folders. 
The tool also allows uses to specify root folders within the archive to achieve the desired formats.

Getting Started
Download the application code (Authored in VS2017 Community) and build to produce the executable.

Prerequisites
VS2017 Community or greater.

Built With
VS2017 Community or greater.
